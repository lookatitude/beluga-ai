# Beluga AI Tutorials

Welcome to the Beluga AI Tutorial library. This collection provides hands-on, practical guides for building advanced AI applications with the Beluga AI Framework.

## Tutorial Categories

### [Foundation](./foundation/README.md)
Core building blocks of the framework including schema modeling, runnables, configuration, and observability.

### [Providers](./providers/README.md)
Deep dives into LLMs, Embeddings, Vector Stores, and Prompts.

### [Higher-Level](./higher-level/README.md)
Advanced patterns for Agents, Chat Models, Memory, Retrievers, Orchestration, Safety, and API Deployment.

### [Voice](./voice/README.md)
Building voice-enabled agents with Speech-to-Text (STT), Text-to-Speech (TTS), and end-to-end Speech-to-Speech (S2S).

### [Other Specialized Tools](./other/README.md)
Messaging (WhatsApp/SMS), Multimodal (Vision/Audio), Document Loaders, and Advanced Text Splitters.

## New to Beluga AI?

If you're just starting out, we recommend following our [Getting Started Series](../getting-started/README.md) first to build a solid foundation.

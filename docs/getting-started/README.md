# Getting Started Tutorial Series

Welcome to the Beluga AI Framework Getting Started Tutorial series! This comprehensive guide will take you from your first LLM call to building production-ready AI applications.

## Tutorial Overview

This series consists of 7 progressive tutorials that build upon each other:

1. **[Your First LLM Call](./01-first-llm-call.md)** - Learn the basics of making LLM API calls
2. **[Building a Simple RAG Application](./02-simple-rag.md)** - Create a retrieval-augmented generation pipeline
3. **[Creating Your First Agent](./03-first-agent.md)** - Build autonomous agents with reasoning capabilities
4. **[Working with Tools](./04-working-with-tools.md)** - Integrate tools to extend agent capabilities
5. **[Memory Management](./05-memory-management.md)** - Add conversation memory to your agents
6. **[Orchestration Basics](./06-orchestration-basics.md)** - Build complex workflows with chains and graphs
7. **[Production Deployment](./07-production-deployment.md)** - Deploy and monitor your AI applications

## Prerequisites

Before starting, ensure you have:

- ✅ **Go 1.24 or later** installed ([Installation Guide](../INSTALLATION.md))
- ✅ **API Keys** for at least one LLM provider:
  - OpenAI: [Get API Key](https://platform.openai.com/api-keys)
  - Anthropic: [Get API Key](https://console.anthropic.com/)
  - Or use Ollama for local models (no API key needed)
- ✅ **Beluga AI Framework** installed:
  ```bash
  go get github.com/lookatitude/beluga-ai
  ```

## Learning Path

### Beginner Path
If you're new to AI development:
1. Start with [Part 1: Your First LLM Call](./01-first-llm-call.md)
2. Progress through each tutorial in order
3. Complete all exercises and examples

### Intermediate Path
If you have experience with LLMs:
1. Review [Part 1](./01-first-llm-call.md) for framework specifics
2. Focus on [Part 2: RAG](./02-simple-rag.md) and [Part 3: Agents](./03-first-agent.md)
3. Explore [Part 4: Tools](./04-working-with-tools.md) for advanced capabilities

### Advanced Path
If you're building production systems:
1. Skip to [Part 6: Orchestration](./06-orchestration-basics.md)
2. Review [Part 7: Production Deployment](./07-production-deployment.md)
3. Reference [Best Practices](../BEST_PRACTICES.md) for production patterns

Short, focused recipes that solve specific problems. Each recipe includes the problem statement, solution code, and explanation. Browse by category or search for a specific topic.

| Category | Recipes | Description |

Each category contains a comprehensive overview plus individual recipe pages. Browse the sidebar categories to find specific topics.

## Recipe Format

Each recipe follows a consistent pattern:

1. **Problem** — What you're trying to solve
2. **Solution** — Working code you can copy
3. **Explanation** — Why it works and trade-offs to consider

<Section title="Get started in <span class='light-text'>minutes</span>" description="Create an AI agent with just a few lines of Go.">

```go
// Create an agent in 10 lines
model, _ := llm.New("openai", llm.ProviderConfig{
    APIKey: os.Getenv("OPENAI_API_KEY"),
    Model:  "gpt-4o",
})

a := agent.New("assistant",
    agent.WithLLM(model),
    agent.WithTools(tools),
)

result, _ := a.Invoke(ctx, "Research GPU pricing trends")
```

# Schema Examples

This directory contains examples demonstrating how to use the Beluga AI Schema package for data structures.

## Examples

### [basic](basic/)

Basic schema operations including creating messages, documents, and multimodal content.

**What you'll learn:**
- Creating different message types (Human, AI, System)
- Creating documents with metadata
- Working with multimodal content (images, voice)
- Message collections

**Run:**
```bash
cd basic
go run main.go
```

## Prerequisites

- Go 1.21 or later
- Beluga AI Framework

## Learning Path

1. Start with `basic` to understand schema fundamentals
2. Try `examples/llm-usage/` to see schemas in LLM operations
3. Explore `examples/rag/simple/` for document usage
4. Check `examples/multimodal/` for multimodal schemas

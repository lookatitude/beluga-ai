# Core Examples

This directory contains examples demonstrating how to use the Beluga AI Core package utilities.

## Examples

### [basic](basic/)

Basic core utilities including error handling, context management, and the Runnable interface.

**What you'll learn:**
- Error handling patterns
- Context management
- Runnable interface
- Options pattern
- Error wrapping

**Run:**
```bash
cd basic
go run main.go
```

## Prerequisites

- Go 1.21 or later
- Beluga AI Framework

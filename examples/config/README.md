# Config Examples

This directory contains examples demonstrating how to use the Beluga AI Config package for configuration management.

## Examples

### [basic](basic/)

Basic configuration operations including loading from environment variables, YAML files, and configuration validation.

**What you'll learn:**
- Creating config managers
- Loading from environment variables
- Loading from YAML files
- Getting and setting configuration values
- Validating configurations

**Run:**
```bash
cd basic
go run main.go
```

## Prerequisites

- Go 1.21 or later
- Beluga AI Framework

## Learning Path

1. Start with `basic` to understand configuration fundamentals
2. Try `examples/llm-usage/config_loader.go` for advanced config patterns
3. Explore package-specific config examples in other packages

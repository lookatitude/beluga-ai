# Memory Examples

This directory contains examples demonstrating how to use the Beluga AI Memory package for conversation history management.

## Examples

### [basic](basic/)

Basic memory operations including buffer memory, window memory, and message history management.

**What you'll learn:**
- Creating different memory types
- Saving and loading conversation context
- Managing message history
- Formatting messages
- Clearing memory

**Run:**
```bash
cd basic
go run main.go
```

## Prerequisites

- Go 1.21 or later
- Beluga AI Framework

## Learning Path

1. Start with `basic` to understand memory fundamentals
2. Try `examples/agents/with_memory` to see memory with agents
3. Explore `examples/rag/with_memory` for RAG with conversation context

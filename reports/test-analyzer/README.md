# Test Analyzer Reports

This directory contains detailed analysis reports generated by the `test-analyzer` tool.

## Files

### Dashboard
- **`dashboard.html`** - Interactive HTML dashboard with visualizations and summaries

### Summary Reports
- **`analysis_summary.md`** - Complete summary across all analyzed packages

### Package-Specific Reports
- **`llms_detailed_report.md`** - Detailed analysis for `pkg/llms` (63 issues, Grade: B-)
- **`memory_detailed_report.md`** - Detailed analysis for `pkg/memory` (321 issues, Grade: D+)
- **`orchestration_detailed_report.md`** - Detailed analysis for `pkg/orchestration` (408 issues, Grade: D)
- **`agents_detailed_report.md`** - Detailed analysis for `pkg/agents` (109 issues, Grade: C+)

## Viewing the Dashboard

Open `dashboard.html` in a web browser to view:
- Overall statistics
- Package comparison tables
- Issue breakdown charts
- Priority recommendations
- Links to detailed reports

## Report Contents

Each detailed report includes:
- Executive summary with issue breakdown
- File-by-file analysis
- Issue categories with code examples
- Specific recommendations
- Test quality metrics
- Package comparisons

## Generating New Reports

To generate new reports, run:

```bash
# Analyze all packages
./bin/test-analyzer --dry-run --output markdown --output-file reports/test-analyzer/analysis_summary.md .

# Analyze specific package
./bin/test-analyzer --dry-run --output markdown --output-file reports/test-analyzer/llms_detailed_report.md pkg/llms
```
